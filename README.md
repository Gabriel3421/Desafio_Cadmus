# Este é o projeto de desafio Cadmus.

O projeto consistem em montar o layout disponibilizado no [Figma](https://www.figma.com/file/NmL2yCH5dwecSHGiuLoLOt/Teste-front?node-id=1%3A8) e consumindo uma [Api](https://chat-cadmus.herokuapp.com/teste/gamers) REST, utilizando das melhores praticas de desenvolvimento frontend.

## Getting Started

### Pré-requisitos

Para usar vc precisará ter o node instalado na sua maquina, bem como o yarn, mas os comandos também podem ser executados utilizando o npm.

### Instalação

Instalando dependências

```
yarn ou npm install
```

Executando o projeto

```
yarn start ou npm start
```

## Decisões/Detalhes de implementação

- Foi utilizado a api de contexto do react para compartilhar informaçoes entre os componentes.

- Por a api retornar no lugar das imagens um link para o google resolvi deixar comentando o código que usa a imagem que vem da api e usar as imagens do próprio layout.
